# 0x04. Files manager

This project is a summary of the ALX back-end trimester: authentication, NodeJS, MongoDB, Redis, pagination and background processing.

## Objective

Build a simple platform to upload and view files:

- User authentication via a token
- List all files
- Upload a new file
- Change permission of a file
- View a file
- Generate thumbnails for images

## Requirements

	->	Allowed editors:
			* vi,
			* vim,
			* emacs,
			* Visual Studio Code
	->	All files will be interpreted/compiled on:
			* Ubuntu 18.04 LTS
				Using:
					* node (version 12.x.x)
	->	All files should end with a new line
	->	A README.md file, at the root of the folder of the project, is mandatory
	->	All code should use:
			* The js extension
	->	All code will be verified against:
			* lint
				using:
					* ESLint
